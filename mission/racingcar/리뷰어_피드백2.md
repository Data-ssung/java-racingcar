List<Car> cars = ready.createCars(input.carName());

Stadium stadium = new Stadium(cars, gameCount);
stadium.playRace();

#### 생성자 체이닝
- `Car` 생성자를 `this`를 이용해서 다시 호출하기 
```java
public Car(String name) {
    this(name,0);
}

public Car(String name, int position) {
    validateName(name);
    this.name = name;
    this.position = position;
}
```
